Computer Vision Algorithm for Action Recognition in Dark Environment

(Final Year Project)

Overview

Research and Development of a two-stage algorithm capable of recognizing human actions in low-light environments. Its primary objective is to contribute to nighttime worker safety monitoring. The project integrates image processing methodologies with the application of computer vision, aiming to achieve an efficient combination of the two techniques. The assessment primarily revolves around evaluating the algorithm's performance concerning processing speed and recognition accuracy in challenging dark conditions.

Methodology/Concept

First Stage Algorithm - Dark Frames Enhancement

CNN=based architecture learn useful information from a set of groundtruth dataset, basically a set of well-lit image, and adjust dark frames to a more visually recptive level.

Second Stage Algorithm - Action Recognition

Apply transfer learning on the pretrained YOLOv7 network to achieve high speed recognition while maintaining the accuracy.
