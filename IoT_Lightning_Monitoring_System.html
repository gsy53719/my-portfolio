Methodology/Concept

Inteferometer (ITF) Mapping Technique

ITF mapping technique is capable for locating a source of signal based on digital interferometric techniques. The development of ITF allows a system receives signals from different antennas that work as an array which being separated at a specific distance of a few wavelengths, which makes the system able to capture the electric field and magnetic field changes caused by the lightning event. In this case, the system requires three of the antennas to be installed perpendicular to each other to form a 90 degrees angle, and the separation between the two antennas is depend on the signal wavelength.

Operation of the System

Three antennas first detect the electromagnetic radiation emitted by the lightning event. Then, the received signal will be transmitted via signal cable to the Pico scope and MATLAB software for signal processing. The analysis of the distance of lightning, the lightning area, the lightning types, and the encoding process will be done here. After that, the processed lightning data will be sent to ThingSpeak server for IoT development, and a HTML webpage will be developed for the user to access the lightning status via the webpage with internet access.
