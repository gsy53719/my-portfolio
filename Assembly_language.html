Controlling Raspberry Pi without Operating System

Overview

Creating embedded programs that can run on a Raspberry Pi computer and control the Pi without the Operating System.

Tools

Raspberry Pi 3 Model B

ARM Flat Assembler Compiler

Methodology/Concept

Bare-metal Programming

Involve the use of assembly language in this case to directly interacts with the hardware without an operating system, enabling direct control over various components such as GPIO pins, timers, and other peripherals.

Results

Raspberry Pi being controlled without OS.
