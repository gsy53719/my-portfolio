Panoptic Segmentation on Chili Fertigation Image for Robot Navigation and Obstacle Avoidance

Overview

Development of a deep learning algorithm to identify the chili fertigation and create a graphical user interface (GUI) for user interaction.

Tools

MATLAB

Methodology/Concept

Naive Approach

Achieve paniptic segmentation by combining both semantic and instance segmentation. Semantic segmentation focus on identifying the different regions or areas in the image based on the objects or elements present. Instance segmentation identify and delineate each individual object separately within the image. Based on this, panoptic segmentation combines the tasks of semantic segmentation and instance segmentation to identify a unified segmentation that covers all elements of the scene, enabling a holistic understanding of the image by considering both the semantics and the instances present.
