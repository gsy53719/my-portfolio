Concept Design of Infrared Camera with Temperature Measuring and Photo Capturing Function

Overview

Concept design of an ARM-based microcontroller system for preventing the spread of COVID-19 and develop program for LPC1708 microcontroller and simulating the data storing and loading process in the microsontroller's registers.

Tools

Fritzing

Keil uVision

Methodology/Concept

Assembly Language Programming

As the project essentially requires to simulate the data storing and loading process in the microcontroller's registers, assembly language is used to direct access the registers. The data transfer process can be simulated in real-time, showing how data is stored, loaded, and manipulated within the microcontroller's registers during the execution of the program.
